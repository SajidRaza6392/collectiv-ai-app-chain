# CollectiVAI Chain – Devnet

This folder will contain configuration files and scripts
for running a local or remote **devnet** of the CollectiVAI Chain.

Planned contents:

- `config/`      – sample configuration files (app.toml, config.toml, genesis.json)
- `scripts/`     – helper scripts to start / reset a devnet
- documentation on how to connect a node and a client (e.g. CollectiVAI App)
